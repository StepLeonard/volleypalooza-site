# 🏐 Volleypalooza Site

## 📌 Project Overview

Volleypalooza is a responsive frontend web application that simulates an event registration platform for a volleyball-themed event.

The goal of this project was to create a clean, user-friendly interface with strong layout structure, responsive design, and interactive form handling. The site focuses on delivering a smooth user experience across all devices while maintaining a modern and visually appealing design.

This project highlights my ability to build polished frontend applications with attention to detail, usability, and performance.
